## How to run locally (for testing purposes :)

1. Install dependencies:
   pip install -r requirements.txt

2. Initialise the database with sample data:
   python init_db.py

3. Start the application:
   python app.py

4. Then you can open your browser at:
   http://127.0.0.1:5000
